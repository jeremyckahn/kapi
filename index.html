Kapi

A keyframing API for the HTML 5 canvas

A JavaScript framework by Jeremy Kahn

  • Github repo - fork 'n follow!

Kapi is a tool that helps you make fun animations on the web. It completely manages animation timing and provides a user-friendly API to define keyframe states. This page contains some demos that show what Kapi can do, and it also serves as a home for the full API documentation. For a tutorial on how to get started with Kapi, please consult the README.

Kapi is free and open source software (MIT License). You are completely free to use, distribute, and modify the code - all I ask is that you credit me appropriately.
