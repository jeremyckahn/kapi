Kapi

A keyframing API for the HTML 5 canvas

A JavaScript framework by Jeremy Kahn

  • Github repo - fork 'n follow!

Kapi is a tool that helps you make fun animations on the web. It completely manages animation timing and provides a user-friendly API to define keyframe states. This page contains some demos that show what Kapi can do. A full API reference is coming very soon (although the source is fully documented if you just can't wait). For a tutorial on how to get started with Kapi, please consult the README.

Kapi is free and open source software (MIT license). You are completely free to use, distribute, and modify the code - all I ask is that you credit me appropriately.

Here's the code that generated what you see above:

var myKapi,
	circle1;

myKapi = kapi(document.getElementById('demo1'), {
	fRate: 30,
	// Set up some basic styles for the canvas
	styles: {
		height: '400px',
		width: '500px',
		background: '#333'
	}
}).play(); // Start the animation

// The actor `circle` is defined in shapes.js
circle1 = myKapi.add(circle, {
	name: 'circle1',
	x: 70,
	y: 70,
	radius: 50,
	color: '#00ffaa',
	easing : 'easeInOutQuint'
}).keyframe(0, {
	// Blank, defining keyframe zero as the same as the initial state
}).keyframe('3s', {
	x: 300,
	y: 300
}).liveCopy('3.25s', 
	'3s'
).keyframe('6s', {
	x: '+=50',
	y: 100
}).liveCopy('8s', 
	0);
